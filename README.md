# Демо. Паттерны проектирования

## Консольный проект

УСТАНОВКА
------------

Из корня проекта с хост-машины запустите:
```bash
make
```

После установки скрипт автоматом переведет в контейнер для ввода команд.

Если контейнеры не запущены:
```bash
make up
make enter
```
